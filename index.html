Basic Popups



onclick attribute of this button:

pop.up('Title Is Here', 'Body Text / Images / Buttons... Are Here');

(Image is from W3Schools)

onclick attribute of this button:

popup_with_image('Title', 'https://www.w3schools.com/tags/img_girl.jpg', 'Caption Here')


popup_with_image(title, img, text) function:

function popup_with_image(title, img, text){
  let contentelem = '<figure><img src="' + img + '"><figcaption>' + text + '</figcaption></figure>';

  pop.up(title, contentelem);
}

Use Basic Popups on your site:

<!-- JQUERY -- >
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

<!-- BASIC POPUPS -->
<link rel="stylesheet" href="popup.css">
<script src="popup.js"></script>

Note: jQuery is required for the popup to work.
